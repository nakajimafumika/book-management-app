# 書籍管理アプリ（社内研修作品）

## 概要
社内研修で開発した書籍管理Webアプリです。  
書籍の登録・一覧表示・貸出・返却・予約などの機能を持ち、研修を通じてJavaやSpring Bootの基礎から実践まで学びました。

## 使用技術
- 言語：Java 21, JavaScript, HTML, CSS
- フレームワーク：Spring Boot 3.5.5
- データベース：MariaDB
- その他：Thymeleaf, QueryDSL

## 主な機能
- 書籍登録・一覧表示
- 書籍貸出／返却・予約機能
- ISBNコードによる書籍情報自動取得（外部API連携）
- CSV・Excel・PDF出力機能
